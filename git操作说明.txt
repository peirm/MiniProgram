写的时候的操作：
git add .
git commit -m "知识点1" // 提交本次操作
git tag 01_知识点1 // 给当前的知识点打一个tag
git tag  // 查看tag
git log // 查看有哪些提交
// git status // 假如有M的话执行(不重要，不执行也可以)
git reset 版本号 // 回退上一个版本(加入提交代码了)
git reset --hard 版本号 // 强制回退上一个版本(假如没有提交代码)

代码全部写完后push到远程
git push --tags
